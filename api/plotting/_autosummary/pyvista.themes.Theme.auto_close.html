pyvista.themes.Theme.auto_close#

property Theme.auto_close: bool[ソース]#

プロットが終了したら，自動的に図を閉じます．

危険

十分に注意して False に設定してください．

例

>>> import pyvista as pv
>>> pv.global_theme.auto_close = False
