pyvista.LookupTable.log_scale#

property LookupTable.log_scale: bool[ソース]#

対数スケールを使用します．

True の場合，ルックアップテーブルは LookupTable.scalar_range に対するログスケールになります．それ以外の場合は線形スケールとなります．

例

ルックアップテーブルにログスケールを使用します．

>>> import pyvista as pv
>>> lut = pv.LookupTable()
>>> lut.log_scale = True
>>> lut.scalar_range = (1, 100)
>>> lut.plot()
