pyvista.plotting.themes.Theme.save#

Theme.save(filename: str) → None[ソース]#

このテーマをjsonファイルにシリアライズします．

before_close_callback はシリアライズできないため，省略します．

パラメータ:
filenamestr

テーマを保存するためのパス． 最後は '.json' で終わる必要があります．

例

テーマをエクスポートしてからロードし直します．

>>> import pyvista as pv
>>> theme = pv.themes.DocumentTheme()
>>> theme.background = 'white'
>>> theme.save('my_theme.json')  
>>> loaded_theme = pv.load_theme('my_theme.json')
