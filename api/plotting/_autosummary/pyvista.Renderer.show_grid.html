show_grid#

Renderer.show_grid(**kwargs)[ソース]#

目盛線と軸ラベルを表示します．

show_bounds のラップされた実装で，グリッド線を使用し，外側のエッジに軸ラベルを表示するようにデフォルトの動作を変更します．これは， matplotlib の grid 機能と類似しています．

パラメータ
**kwargsdict, optional

その他のキーワード引数については， Renderer.show_bounds() を参照してください．

戻り値
vtk.vtkAxesActor

境界アクター．

例

>>> import pyvista
>>> from pyvista import examples
>>> pl = pyvista.Plotter()
>>> _ = pl.add_mesh(examples.download_guitar())
>>> _ = pl.show_grid()
>>> pl.show()
