smooth_shading#

property DefaultTheme.smooth_shading: bool#

デフォルトの smooth_shading のパラメーターを返すか，設定します．

例

グローバル smooth_shading パラメータのデフォルトを True に設定します．

>>> import pyvista
>>> pyvista.global_theme.smooth_shading = True
