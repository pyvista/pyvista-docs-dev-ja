pyvista.RenderWindowInteractor.add_pick_obeserver#

RenderWindowInteractor.add_pick_obeserver(observer)[ソース]#

ピックイベントの終了時にコールバックするオブザーバーを追加します．

バージョン 0.42.2 で非推奨: この関数は非推奨です．代わりに pyvista.plotting.RenderWindowInteractor.add_pick_observer() を使用してください．

パラメータ:
observercallable()

ピックイベントが終了したときに呼び出されるオブザーバー関数．
