pyvista.LookupTable.hue_range#

property LookupTable.hue_range: tuple | None[ソース]#

色相範囲を返すか設定します．

この範囲はカスタムカラーマップを作成するときにのみ使用され， LookupTable.cmap でカラーマップを設定した場合は None を返します．

これは，既存のカラーマップをクリアし，設定時にルックアップテーブルの新しい値を作成します．

例

色相の範囲を設定します．これにより，カラーマップを設定せずにルックアップテーブルを作成することができます．

>>> import pyvista as pv
>>> lut = pv.LookupTable()
>>> lut.hue_range = (0, 0.1)
>>> lut.plot()

別のカラーマップを作成します．

>>> import pyvista as pv
>>> lut = pv.LookupTable()
>>> lut.hue_range = (0.5, 0.8)
>>> lut.plot()
