pyvista.plotting.charts.Axis#

class Axis(label='', range=None, grid=True, *, _wrap=None)[ソース]#

2Dチャートで使用されるVTK AxisのPythonicインターフェースです．

パラメータ:
labelstr, default: ""

軸のラベル．

rangesequence[float], optional

軸の範囲，この軸に表示される最小値と最大値を指定します．これを None 以外の有効な値に設定すると，この軸の動作は 'fixed' に変更されます．これを None に設定すると，この軸の動作を 'auto' に変更します．

gridbool, default: True

この軸のグリッド線の表示/非表示を切り替えるフラグです．

アトリビュート:
penPen

軸の描画に使用するペン．

grid_penPen

グリッド線の描画に使用するペン．

メソッド

Axis.toggle()

軸の表示を切り替えます．

アトリビュート

Axis.behavior

軸のスケーリング動作を設定します．

Axis.grid

軸のグリッドラインの表示を返すか設定します．

Axis.label

軸ラベルを返すか設定します．

Axis.label_size

軸ラベルフォントのサイズを返すか設定します．

Axis.label_visible

軸のラベルの表示を返すか設定します．

Axis.log_scale

この軸に対数スケールを使用するかどうかを示すフラグ．

Axis.margin

軸余白を返すか設定します．

Axis.range

軸範囲を返すか設定します．

Axis.tick_count

この軸に描画される目盛りの数を返すか設定します．

Axis.tick_label_size

軸目盛りラベルフォントのサイズを返すか設定します．

Axis.tick_labels

この軸目盛りラベルを返すか設定します．

Axis.tick_labels_offset

この軸目盛りラベルのオフセットを返すか設定します．

Axis.tick_labels_visible

この軸目盛りラベルの表示を返すか設定します．

Axis.tick_locations

この軸目盛り位置を返すか設定します．

Axis.tick_size

このグラフの軸目盛りのサイズを返すか設定します．

Axis.ticks_visible

この軸目盛りの表示を返すか設定します．

軸の表示を返すか設定します．
