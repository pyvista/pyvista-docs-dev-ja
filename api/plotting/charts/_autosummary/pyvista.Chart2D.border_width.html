border_width#

property Chart2D.border_width#

チャートの境界線幅を返すか設定します．

例

赤色の太い破線の境界線を持つ2Dチャートを作成します．

>>> import pyvista
>>> chart = pyvista.Chart2D()
>>> plot = chart.line([0, 1, 2], [2, 1, 3])
>>> chart.border_color = 'r'
>>> chart.border_width = 5
>>> chart.border_style = '--'
>>> chart.show()
