プロットAPIリファレンス#

プロットモジュールAPIリファレンス．これらのプロットモジュールは，PyVistaのすべてのプロット機能の基礎となります．

Actor([mapper, prop, name])

vtkActor のラップ．

Axes([show_actor, actor_scale, line_width, ...])

VTK AxesクラスのPyVistaラッパー．

AxesActor()

vtkAxesActor の軸のアクターラッパーです．

CameraPosition(position, focal_point, viewup)

カメラ位置の属性を保持するためのコンテナ．

CubeAxesActor(camera[, minor_ticks, ...])

vtkCubeAxesActor のラップ．

DataSetMapper([dataset, theme])

_vtk.vtkDataSetMapper をラップします．

LookupTable([cmap, n_values, flip, values, ...])

スカラからRGBAへのマッピングテーブル．

Plotter(*args, **kwargs)

vtkメッシュまたは無感覚な配列を表示するためのオブジェクトのプロット．

Property([theme, interpolation, color, ...])

vtkProperty をラップして pythonically に公開します．

Renderer(parent[, border, border_color, ...])

Rendererクラス．

RenderWindowInteractor(plotter[, ...])

vtk.vtkRenderWindowInteractor をラップします．

Timer(max_steps, callback)

Timer クラス．

plotting.mapper._BaseMapper(*args, **kwargs)

他のマッパーと共通するメソッドを持つベースマッパーです．

plotting.opts.InterpolationType(value)

照明補間の種類．

plotting.opts.RepresentationType(value)

モデルが持つことのできる表現の種類．

plotting.opts.ElementType(value)

要素幾何学の種類．

plotting.volume.Volume()

VTKボリュームのラッパークラスです．

plotting.volume_property.VolumeProperty([...])

VTK クラス vtkVolumeProperty をラッピングします．

複合プロットリファレンス#

これらのクラスは pyvista.MultiBlock データセットをプロットするときに使用されます．

CompositePolyDataMapper([dataset, theme, ...])

複合 PolyData マッパー．

CompositeAttributes(mapper, dataset)

ブロックの属性．

BlockAttributes(block, attr)

ブロックの属性を設定するために使用するブロック属性．

チャートAPI#

Charts APIのリファレンスです．これらの専用クラスは，プロットウィンドウにチャートを埋め込むために使用することができます．なお，チャートの全機能を利用するには，少なくともVTKのバージョン9.3が必要です．しかし，ほとんどのコンポーネントは，古いバージョンでも問題なく動作します．

ウィジェットAPI#

pyvista.Plotter クラスは pyvista.plotting.widgets.WidgetHelper クラスで記述されているすべてのウィジェットメソッドを継承しています．詳しくは ウィジェット の例を参照してください．

plotting.widgets.WidgetHelper(*args, **kwargs)

ウィジェットを管理する内部クラス．

plotting.widgets.AffineWidget3D(plotter, actor)

3D affine 変換ウィジェット．

ピッキングAPI#

pyvista.Plotter クラスは pyvista.plotting.picking.PickingHelper クラスで記述されているすべてのピッキングメソッドを継承しています．

plotting.picking.PickingHelper(*args, **kwargs)

ピッキングヘルパーメソッドを含む内部コンテナクラス．
