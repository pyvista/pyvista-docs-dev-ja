pyvista.Camera.clipping_range#

property Camera.clipping_range[ソース]#

クリッピング・プレーンの位置を返すか設定します．

Clipping Planesとは，投影方向に沿った近景と遠景のClipping Planesのことです．

例

>>> import pyvista as pv
>>> pl = pv.Plotter()
>>> pl.camera.clipping_range
(0.01, 1000.01)
>>> pl.camera.clipping_range = (1, 10)
>>> pl.camera.clipping_range
(1.0, 10.0)
