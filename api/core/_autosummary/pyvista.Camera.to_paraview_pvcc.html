pyvista.Camera.to_paraview_pvcc#

Camera.to_paraview_pvcc(filename: str | Path)[ソース]#

カメラパラメータをParaviewカメラファイル (拡張子.pvcc) に書き込みます．

パラメータ:
filenamepython:str または pathlib.Path

Paraviewカメラファイル (.pvcc) へのパス．

例

>>> import pyvista as pv
>>> pl = pv.Plotter()
>>> pl.camera.to_paraview_pvcc("camera.pvcc")
