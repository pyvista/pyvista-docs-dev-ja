pyvista.PolyData.verts#

property PolyData.verts: ndarray[ソース]#

頂点セルを取得します．

戻り値:
numpy.ndarray

頂点セルのインデックスの配列．

例

点群ポリデータを作成し，その頂点セルを返します．

>>> import pyvista
>>> import numpy as np
>>> points = np.random.random((5, 3))
>>> pdata = pyvista.PolyData(points)
>>> pdata.verts
array([1, 0, 1, 1, 1, 2, 1, 3, 1, 4])

頂点セルを設定します． サーフェスのメッシュと追加の頂点の両方を1つのプロットにしていることに注目してください．

>>> mesh = pyvista.Plane(i_resolution=3, j_resolution=3)
>>> mesh.verts = np.vstack(
...     (
...         np.ones(mesh.n_points, dtype=np.int64),
...         np.arange(mesh.n_points),
...     )
... ).T
>>> mesh.plot(
...     color='tan',
...     render_points_as_spheres=True,
...     point_size=60,
... )
