pyvista.DataSetFilters#

class DataSetFilters(*args, **kwargs)[ソース]#

任意のvtkDataSetに適用できる共通フィルタのセット．

メソッド

DataSetFilters.align(target[, ...])

データセットを別のデータセットに位置合わせします．

DataSetFilters.align_xyz(*[, centered, ...])

データセットをx-y-z軸に位置合わせします．

DataSetFilters.bounding_box([box_style, ...])

このデータセットの境界ボックスを返します．

DataSetFilters.cell_centers([vertex, ...])

このデータセットのセルの中心に点を生成します．

DataSetFilters.cell_data_to_point_data([...])

データから日付へのセル変換

DataSetFilters.clip([normal, origin, ...])

原点と法線を指定して，データセットを平面でクリップします．

DataSetFilters.clip_box([bounds, invert, ...])

境界によって定義された境界ボックスによってデータセットをクリップします．

DataSetFilters.clip_scalar([scalars, ...])

データセットをスカラーでクリップします．

DataSetFilters.clip_surface(surface[, ...])

pyvista.PolyData サーフェスメッシュを使用して任意のメッシュタイプをクリップします．

DataSetFilters.color_labels([colors, ...])

Add RGB(A) scalars to labeled data.

DataSetFilters.compute_boundary_mesh_quality(*)

メッシュの境界面のメトリックを計算します。

DataSetFilters.compute_cell_quality([...])

メッシュのセルごとに (ジオメトリ) 精度の関数を計算します．

DataSetFilters.compute_cell_sizes([length, ...])

0D (頂点数) ，1 D (長さ) ，2 D (面積) ，および3 D (体積) セルのサイズを計算します．

DataSetFilters.compute_derivative([scalars, ...])

点/セル スカラーフィールドの派生ベースの量を計算します．

DataSetFilters.compute_implicit_distance(surface)
