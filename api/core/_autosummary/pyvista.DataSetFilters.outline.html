pyvista.DataSetFilters.outline#

DataSetFilters.outline(generate_faces=False, progress_bar=False)[ソース]#

入力データセットの全エクステントのアウトラインを作成します．

パラメータ:
generate_facesbool, default: False

直方体のソリッド面を生成します．デフォルトでは無効になっています．

progress_barbool, default: False

進行状況を示す進行状況バーを表示します．

戻り値:
pyvista.PolyData

元のデータセットのアウトラインを含みますメッシュ．

例

球体の輪郭を生成し，プロットします． これは事実上，メッシュの (x, y, z) 境界を表しています．

>>> import pyvista
>>> sphere = pyvista.Sphere()
>>> outline = sphere.outline()
>>> pyvista.plot([sphere, outline], line_width=5)

このフィルターを使用したその他の例については， 共通フィルタの使用 を参照してください．
