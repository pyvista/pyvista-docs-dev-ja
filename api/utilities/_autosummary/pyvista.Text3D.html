pyvista.Text3D#

Text3D(string, depth=0.5)[ソース]#

文字列から3 Dテキストを作成します．

パラメータ:
stringstr

3Dテキストを生成するための文字列．

depthfloat, default: 0.5

テキストの奥行き．

戻り値:
pyvista.PolyData

3Dテキストメッシュ．

例

>>> import pyvista
>>> text_mesh = pyvista.Text3D('PyVista')
>>> text_mesh.plot(cpos='xy')
