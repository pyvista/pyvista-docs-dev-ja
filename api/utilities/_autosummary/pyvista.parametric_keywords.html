pyvista.parametric_keywords#

parametric_keywords(parametric_function, min_u=0, max_u=6.283185307179586, min_v=0.0, max_v=6.283185307179586, join_u=False, join_v=False, twist_u=False, twist_v=False, clockwise=True)[ソース]#

パラメトリック関数にキーワード引数を適用します．

パラメータ:
parametric_functionvtk.vtkParametricFunction

メッシュを生成するパラメトリック関数．

min_ufloat, optional

U値の最小値．

max_ufloat, optional

U値の最大値．

min_vfloat, optional

V値の最小値．

max_vfloat, optional

V値の最大値．

join_ubool, optional

最初の3角形のストリップを最後の3角形のストリップにU方向のねじれで結合します．

join_vbool, optional

最初の3角形の条片を，V方向にねじった最後の条片に結合します．

twist_ubool, optional

最初の3角形のストリップを最後の3角形のストリップにU方向のねじれで結合します．

twist_vbool, optional

最初の3角形の条片を，V方向にねじった最後の条片に結合します．

clockwisebool, optional

3角形の条片を形成する頂点の順序を決定します．
