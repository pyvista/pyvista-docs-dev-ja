pyvista.transform_vectors_sph_to_cart#

transform_vectors_sph_to_cart(theta, phi, r, u, v, w)[ソース]#

球面座標 (r, phi, theta) からデカルト座標 (z, y, x)にベクトルを変換します．

地球科学でよく使われる，配列の軸の "逆" 順序に注意．

パラメータ:
thetaarray_like[float]

形状 (M,) の度単位の方位角度 [0, 360] ．

phiarray_like[float]

形状 (N,) の度単位の極(天頂)角度 [0, 180] ．

rarray_like[float]

形状 (P,) の原点からの距離(半径)．

uarray_like[float]

形状 (P, N, M) のベクトルのX成分．

varray_like[float]

形状 (P, N, M) のベクトルのY成分．

warray_like[float]

形状 (P, N, M) のベクトルのZ成分．

戻り値:
u_t, v_t, w_tnumpy.ndarray

それぞれ変換されたx-，y-，z-成分の配列．
