pyvista.Circle#

Circle(radius=0.5, resolution=100)[ソース]#

XY平面上に半径で定義されたPolyDataの円を1つ作成します．

パラメータ:
radiusfloat, default: 0.5

円の半径．

resolutionint, default: 100

円上の点の数．

戻り値:
pv.PolyData

円のメッシュ．

備考

バージョン 0.38.0 で変更: バージョン0.38以前では，この方法は誤った結果をもたらし，一貫性のないエッジの長さと重複した点を生成していましたが，現在は修正されています．

例

>>> import pyvista as pv
>>> radius = 0.5
>>> circle = pv.Circle(radius)
>>> circle.plot(show_edges=True, line_width=5)
