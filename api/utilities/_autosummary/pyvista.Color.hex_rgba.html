hex_rgba#

property Color.hex_rgba: str#

色の値をRGBAの16進数で取得します．

例

半分の不透明度で青色を作成します．

>>> import pyvista
>>> c = pyvista.Color("blue", default_opacity="#80")
>>> c
Color(name='blue', hex='#0000ff80')
>>> c.hex_rgba
'#0000ff80'

RGBAの16進数値で透明な赤色を作成します．

>>> c = pyvista.Color("0xff000040")
>>> c
Color(name='red', hex='#ff000040')
>>> c.hex_rgba
'#ff000040'
