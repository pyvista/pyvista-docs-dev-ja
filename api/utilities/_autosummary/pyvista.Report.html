pyvista.Report#

class Report(additional=None, ncol=3, text_width=80, sort=False, gpu=True)[ソース]#

PyVistaソフトウェア環境レポートを作成します．

パラメータ:
additionalsequence[types.ModuleType], sequence[str]

出力情報に追加するパッケージまたはパッケージ名のリスト．

ncolint, default: 3

html テーブルの package-column の数; mode='HTML' または mode='html' の場合のみ有効です．

text_widthint, default: 80

HTML以外の表示モードでの文字幅を指定します．

sortbool, default: False

パッケージをアルファベット順に並べます．

gpubool, default: True

GPU に関する情報を収集します．デフォルトは True ですが，レンダリングに問題がある場合は False を渡すと，安全にレポートを生成することができます．
