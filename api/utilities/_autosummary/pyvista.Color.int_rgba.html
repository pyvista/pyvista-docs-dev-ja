pyvista.Color.int_rgba#

property Color.int_rgba: Tuple[int, int, int, int][ソース]#

色の値を RGBA integer のタプルとして取得します．

例

半分の不透明度で青色を作成します．

>>> import pyvista
>>> c = pyvista.Color("blue", opacity=128)
>>> c
Color(name='blue', hex='#0000ff80', opacity=128)
>>> c.int_rgba
(0, 0, 255, 128)

整数の RGBA シーケンスを使って，透明な赤色を作成します．

>>> c = pyvista.Color([255, 0, 0, 64])
>>> c
Color(name='red', hex='#ff000040', opacity=64)
>>> c.int_rgba
(255, 0, 0, 64)
