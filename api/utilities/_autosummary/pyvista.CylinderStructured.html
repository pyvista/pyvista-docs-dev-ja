CylinderStructured#

CylinderStructured(radius=0.5, height=1.0, center=(0.0, 0.0, 0.0), direction=(1.0, 0.0, 0.0), theta_resolution=32, z_resolution=10)[ソース]#

pyvista.StructuredGrid として円柱メッシュを作成します．

エンドキャップは開いたままです．これにより， radius に単一の値が指定された場合は表面メッシュが作成され， radius 引数に複数の半径がリスト/配列として指定された場合は3 Dメッシュが作成されます．

パラメータ
radiusfloat, sequence, optional

円柱の半径です．直列の場合は，セルの半径方向の座標を， radius で指定された値の範囲で記述します．

heightfloat, optional

Z軸に沿った円柱の高さ

centersequence

[x, y, z] 中のセントロイドの位置

directionsequence

[x, y, z] の方向円柱Z軸．

theta_resolutionint, optional

円柱の円形面上の点の数． radius が反復可能である場合は無視されます．

z_resolutionint, optional

円柱の高さ (Z軸) に沿った点の数．

戻り値
pyvista.StructuredGrid

構造化円柱．

例

デフォルトの円柱．

>>> import pyvista
>>> mesh = pyvista.CylinderStructured()
>>> mesh.plot(show_edges=True)

内側の半径が1，外側の半径が2の構造化された円筒で，5つのセグメントがあります．

>>> import numpy as np
>>> mesh = pyvista.CylinderStructured(radius=np.linspace(1, 2, 5))
>>> mesh.plot(show_edges=True)
