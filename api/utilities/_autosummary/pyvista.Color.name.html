pyvista.Color.name#

property Color.name: str | None[ソース]#

色名を返します．

戻り値:
str | None

この色に名前がある場合は色の名前，それ以外の場合は None です．

備考

対応色一覧は下表をご参照ください．

名称

16進数値

例

>>> import pyvista as pv
>>> c = pv.Color("blue", default_opacity=0.5)
>>> c
Color(name='blue', hex='#0000ff80', opacity=128)
