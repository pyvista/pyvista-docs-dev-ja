lines_from_points#

lines_from_points(points, close=False)[ソース]#

点の配列を指定して，接続された線分セットを作成します．

パラメータ
pointsnp.ndarray

接続されたセグメントの頂点を表すポイント．たとえば，2つの線分セグメントは np.array([[0, 0, 0], [1, 0, 0], [1, 1, 0]]) のように表されます．

closebool, optional

True の場合，線分セグメントを閉じてループを作成します．

戻り値
pyvista.PolyData

ラインやセルを使ったPolyData．

例

>>> import numpy as np
>>> import pyvista
>>> points = np.array([[0, 0, 0], [1, 0, 0], [1, 1, 0]])
>>> poly = pyvista.lines_from_points(points)
>>> poly.plot(line_width=5)
