Readers#

PyVistaには，データファイルの読み込みをより細かく制御できるクラスベースのリーダがあります． これらのクラスは，ファイルからデータセットを読み込む際に，より細かい制御を可能にします． サポートしているファイルタイプの一覧は， pyvista.get_reader() を参照してください．

また，リーダークラスを使った完全な例として， リーダーによるデータの読み込み を参照してください．

get_reader(filename[, force_ext])

データファイルの読み取りをきめ細かく制御するためのリーダーを取得します．

継承クラス#

pyvista.BaseReader は，すべてのサブリーダーによって継承されます． このクラスは，ファイル名を設定してデータを読み取るという，すべてのリーダーの基本的な機能を持っています．

PointCellDataSelection は，ポイントやセルの配列に関連するデータの検査や設定をサポートするリーダーによって継承されます．

TimeReader は，読み取りのための時間やイテレーションの検査や設定をサポートするリーダーによって継承されます．

BaseReader(*args, **kwargs)

Base Readerクラス．

PointCellDataSelection()

データ配列の選択をサポートするリーダー用のMixinです．

TimeReader()

時間をサポートするリーダーのための抽象クラスです．
