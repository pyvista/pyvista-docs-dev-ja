pyvista.examples.examples.load_explicit_structured#

load_explicit_structured(dimensions=(5, 6, 7), spacing=(20, 10, 1))[ソース]#

Load a simple explicit structured grid.

Parameters
dimensionstuple(int), optional

Grid dimensions. Default is (5, 6, 7).

spacingtuple(int), optional

Grid spacing. Default is (20, 10, 1).

Returns
pyvista.ExplicitStructuredGrid

An explicit structured grid.

Examples

>>> from pyvista import examples
>>> grid = examples.load_explicit_structured()
>>> grid.plot(show_edges=True)
