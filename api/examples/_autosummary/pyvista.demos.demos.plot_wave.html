pyvista.demos.demos.plot_wave#

plot_wave(fps=30, frequency=1, wavetime=3, interactive=False, notebook=None)[ソース]#

3Dの動く波をレンダリングウィンドウにプロットします．

パラメータ:
fpsint, default: 30

非表示にする最大秒間フレーム数．

frequencyfloat, default: 1.0

1秒あたりの波の周期 (Hz) ．

wavetimefloat, default: 3.0

希望する総表示時間を秒単位で指定します．

interactivebool, default: False

波の動きが始まる前に，カメラの位置を設定することができます．

notebookbool, optional

True の場合，作成されたプロットはjupyterノートブック内に配置されます．jupyterコンソールがアクティブであると仮定します．

戻り値:
numpy.ndarray

最後のフレームでのポイントの位置．

例

>>> from pyvista import demos
>>> out = demos.plot_wave()
