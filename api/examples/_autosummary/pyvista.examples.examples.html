pyvista.examples.examples#

PyVistaに同梱されていて，ダウンロードする必要のない内蔵サンプル．

例#

>>> from pyvista import examples
>>> mesh = examples.load_ant()
>>> mesh.plot()

関数

load_airplane()

ply飛行機メッシュをロードします．

load_ant()

ply蟻メッシュをロードします．

load_channels()

地下に一様な格子状の河道を読み込みます．

load_explicit_structured([dimensions, spacing])

単純な明示的構造のグリッドを読み込みます．

load_globe()

地球儀のソースを読み込みます．

load_globe_texture()

地球儀ソースに適用可能なvtk.vtkTextureを読み込みます．

load_hexbeam()

サンプルのUnstructuredGridを読み込みます．

load_hydrogen_orbital([n, l, m, zoom_fac])

水素の波動関数を pyvista.ImageData にロードします．

load_nut()

サンプルのナットメッシュをロードします．

load_random_hills()

ランダムな丘のおもちゃの例を作成します．

load_rectilinear()

サンプルのユニフォームグリッドを読み込みます．

load_sphere()

球plyメッシュをロードします．

load_sphere_vectors()

ノードに定義された渦巻き状のベクトルフィールドを持つ球体の例を作成します．

load_spline()

スプラインメッシュの例を読み込みます．

load_structured()

シンプルなStructuredGridを読み込みます．

load_tetbeam()

4面体セルのみを含みますサンプルUnstructuredGridをロードします．

load_uniform()

サンプルのユニフォームグリッドを読み込みます．
