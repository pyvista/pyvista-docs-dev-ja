pyvista.examples.downloads#

様々なソースから集められたダウンロード可能なデータセット．

一度ダウンロードしたデータセットはローカルに保存されるため，迅速に再利用することができます．

ファイルはすべて pyvista/vtk-data でホストされており， download_file 関数を使用してダウンロードします．サンプルデータリポジトリにファイルを追加する場合は，このページでレンダリングされる download-<dataset> メソッドを追加してください．

ファイルやデータセットのメタデータを含むデータセットの詳細情報については、 データセットギャラリー を参照のこと。

例#

>>> from pyvista import examples
>>> mesh = examples.download_saddle_surface()
>>> mesh.plot()

関数

delete_downloads()

ダウンロードした例題をすべて削除して容量を確保するか，ファイルを更新します．
