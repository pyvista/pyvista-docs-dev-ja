Installation#

The only prerequisite for installing PyVista is Python itself. If you don’t have Python yet and want the simplest way to get started, we recommend you use the Anaconda Distribution.

Working with conda?

PyVista is available on conda-forge.

conda install -c conda-forge pyvista
Prefer pip?

PyVista can be installed via pip from PyPI.

pip install pyvista
In-depth instructions?

Installing a specific version? Installing from source? Check the Installation page.

Support#

For general questions about the project, its applications, or about software usage, please create a discussion in pyvista/discussions where the community can collectively address your questions. You are also welcome to join us on Slack or send one of the developers an email. The project support team can be reached at info@pyvista.org.
